Computes the cosinus sign in a "right-handed" coordinate system (i.e. clockwise angle values). cossign(x, y, z) > 0 iff angle xyz is acute.

http://en.wikipedia.org/wiki/Dot_product#Geometric_definition

A.B = ||A|| ||B|| cos t
A.B = A0 B0 + A1 B1
A = [y[0] - x[0], y[1] - x[1]]
B = [y[0] - z[0], y[1] - z[1]]
A.B = (y[0] - x[0]) * (y[0] - z[0]) + (y[1] - x[1]) * (y[1] - z[1])
